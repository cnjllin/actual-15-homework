
#作业：
#用js实现登录 注册，查询，删除，更新模块  
    

##项目分析：完善 用户管理（个人信息，用户列表） 资产管理（机房管理，机柜管理）

*  1首页：
     *首页存在仪表盘 用户管理两个模块
     *客户如没有登录，则跳转到登录界面
* 2注册：
     *  在userlist选择注册，跳转到注册页面，输入相关信息
     * 如果用户名存在，返回注册页面并提示用户名已存在
     *  如用户名不存在，跳转到登录页面

* 3登录：
     * 输入信息正确，如信息不对，在登陆界面提示错误
     * 如用户是管理员，则跳转到主界面，如是用户则跳转到个人信息
* 4userlist：
     * 在userlist面有注册，更新，删除，查询四大模块
     * 点击功能模块，分别跳转到对应页面

* 5删除：
     * 在userlist界面点击删除，并返回userlist界面
* 6更新：
     * 在userlist界面点击删除，并返回userlist界面
     * 输入修改信息，修好后跳转到userlist界面
*  7查询：
     *在userlist界面内点击更新，跳转到个人信息界面
