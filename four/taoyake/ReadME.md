###目录定义

```
homework_01                  : 这个目录为flask注册登录作业的目录
	homework_01/templates    :  存放三个html界面，login.html、register.html、userlist.html
	user.py                  :  获取用户信息和验证登录的函数模块
	userinfo.txt             :  存放用户和密码的文件
	web.py                   :  主程序，用于判断用户登录和注册，以及用户信息展示的函数模块
homework_top10				 :   这个目录为获取nginx日志top10ip作业
	templates	             :   页面展示top10的html
	top10.py                 :    渲染nginx top10的代码
项目需求文档.md                :   项目需求分析文档

```
