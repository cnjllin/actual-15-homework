
##作业：
用数据库实现客户登录注册，并添加查询，删除，更新模块 



##项目分析：

* 首页：
    *  首页存在登录注册两个模块
    *  点击登录跳转到登录界面，点击注册跳转到注册界面
* 注册：
     * 首页选择注册，跳转到注册页面，输入相关信息
     * 如果用户名存在，返回注册页面并提示用户名已存在
     * 如用户名不存在且两次密码输入一致，跳转到登录页面
* 登录：
     * 首页选择登录界面，跳转到登录界面，输入登录信息
     * 输入信息正确，跳转到userlist界面
     * 输入用户名或密码有误，提示错误信息
* userlist：
     * 查询数据库列表信息，并渲染到userlist界面
     * 添加更新，删除，查询功能
     * 点击功能模块，分别跳转到对应页面
* 删除：
     * 在userlist界面点击删除，并返回userlist界面
* 更新:
     * 在userlist界面内点击更新，跳转到更新界面（修改密码）
     * 输入修改信息，修好后跳转到userlist界面
* 查询：
     * 在userlist界面内点击更新，跳转到查询界面
