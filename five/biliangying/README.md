## 项目目标

* 实现用户注册登录功能，将数据写入数据库中，管理用户信息

## 基本用户功能
* 注册、登录
* 删除用户
* 修改密码
## 项目实现
* 欢迎界面
    * 欢迎界面，用户选择登录和注册，利用href链接到注册和登录界面	
* 注册
    * 判断用户名是否存在，存在提示错误信息
    * 判断密码是否一致，不一致提示错误信息
    * 用户注册成功redirect到登录界面
* 登录
    * 判断用户名是否存在，不存在提示错误信息
    * 判断密码是否正确，错误提示错误信息
    * 用户登录成功redirect到首页用户列表
* 首页
    * 用户列表，用户可对信息进行删除和修改密码操作
*  修改密码
   * 判断密码是否一致，不一致提示错误信息
   * 一致回到首页
* 删除用户
  * 删除成功后回到首页
